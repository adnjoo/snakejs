# Snake Game

A simple Snake Game built with React and Vite.

<img src='./public/scrn.png' width='300'>

## Setup

1. Install dependencies:  
   ```bash
   npm install
   ```

2. Start the development server:  
   ```bash
   npm run dev
   ```

3. Build for production:  
   ```bash
   npm run build
   ```

Enjoy the game!
